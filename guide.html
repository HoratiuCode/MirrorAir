MirrorAir Guide

Install and fix install problems.

Use this page if Android blocks the APK, stops the package installer, or says the app could not be installed.

How To Install

  1. Use an Android device running Android 6 or newer.
  2. Tap Download APK and wait for MirrorAir-v2.apk to finish downloading.
  3. If Android blocks the file, allow Install unknown apps for your browser or the Files app.
  4. Open the downloaded APK from the Android Files or Downloads app.
  5. Install MirrorAir and then open it on the same Wi-Fi network as your Mac.

If Install Fails

  1. Delete every older MirrorAir.apk or MirrorAir-v2.apk file from Downloads before trying again.
  2. Uninstall any older MirrorAir or MirrorNode app already installed on the device.
  3. Download the APK again instead of reusing an older file from browser history.
  4. If Android says There was a problem parsing the package, make sure the device is Android 6 or newer and the file finished downloading completely.
  5. If Android says App not installed, try opening the APK from the Files app instead of the browser popup.
  6. If the Android package installer keeps stopping, restart the phone once and try again from the Files app.

Direct Install From Mac

  1. Enable Developer options and USB debugging on the Android device.
  2. Connect the phone or tablet to your Mac with USB and approve the computer on the device.
  3. From the project folder, run scripts/install-debug-apk.sh.
  4. This uses adb install -r, which bypasses the Android package installer popup path.

After Install

  1. Open MirrorAir on Android.
  2. Press Start Receiver.
  3. On your Mac, open Screen Mirroring or AirPlay and look for MirrorAir.
  4. If the Mac sees the device but cannot connect, the app build is installed correctly and the remaining issue is receiver compatibility, not APK installation.
