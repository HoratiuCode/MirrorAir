Direct Download

Host the APK in downloads/ and download it directly on the Android device you want to mirror to.

Install

Download the APK, allow app installs from your browser if Android asks, then open MirrorAir on the Android device you want to use as the Mac display target.

If install fails, delete any older downloaded APK file first, uninstall older MirrorAir or MirrorNode builds, then download the versioned package again.
