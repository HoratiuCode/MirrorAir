How it works

The Android app starts a foreground receiver service, advertises itself on the local Wi-Fi network, and is designed to host a native AirPlay-compatible mirroring engine.

Android Receiver

The APK launches the receiver shell, foreground service, and Android video surface used by the native stack.

Current State

This build is experimental. Discovery, native startup, and Android video decode are wired in, but device compatibility still varies.

Static Download Page

Publish this docs/ folder on GitHub Pages and host your built APK directly from the repo.

Install

Download the APK, allow app installs from your browser if Android asks, then open MirrorAir on the Android device you want to use as the Mac display target.

If install fails, delete any older downloaded APK file first, uninstall older MirrorAir or MirrorNode builds, then download the versioned package again.
