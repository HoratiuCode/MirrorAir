MirrorAir

Mirror your Mac screen to Android.

A minimal Android receiver project with AirPlay-style discovery, a fullscreen receiver surface, and a download page ready for APK hosting.

How it works

The Android app starts a foreground receiver service, advertises itself on the local Wi-Fi network, and is designed to host a native AirPlay-compatible mirroring engine.

Android Receiver

Minimal black UI, configurable receiver name, auto-start, multicast support, and DNS-SD discovery.

Mac Detection

The app publishes AirPlay-style network services so a Mac can see it as a candidate receiver target.

Static Download Page

Publish this docs/ folder on GitHub Pages and host your built APK directly from the repo.

Install

  1. Build the Android app in Android Studio.
  2. Run scripts/prepare-downloads.sh.
  3. Publish the docs/ folder with GitHub Pages.
  4. Open the site on your Android device and download the APK.
